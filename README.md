Proyecto cuyo objetivo es medir tiempos de ejecución de distintos algoritmos. 

Para poder usar este proyecto correctamente deberán de seguir los siguientes pasos: 

1. Clonar el repositorio
2. Situar la terminal en la carpeta de la practica correspondiente: cd "...\Practica 1\practica_01"
3. Compilar el código: javac -d out -sourcepath src/main/java src/main/java/es/Main/Main.java
4. Ejecutar el comando: java -cp out es.Main.Main
